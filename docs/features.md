# LedgeGuard - Feature Roadmap

This document catalogs the features of our production-grade finance backend.

## Phase 1: Authentication & Access Control (Status: [IN PROGRESS])
- [ ] **JWT Auth with Refresh Tokens**: Secure token-based access.
- [ ] **Role-Based Access Control (RBAC)**: Viewer, Analyst, and Admin roles.
- [ ] **User Management**: Creation and status (Active/Inactive) handling.

## Phase 2: Financial Records (Status: [PLANNED])
- [ ] **Transaction CRUD**: Income/Expense records.
- [ ] **Transaction Filtering**: Date, Category, and Type filters.
- [ ] **Soft Delete Support**: Secure record deletion.

## Phase 3: Analytics & Dashboard (Status: [PLANNED])
- [ ] **Aggregation Engine**: Total Income, Balance, Category-wise totals.
- [ ] **Monthly Trends**: Time-series data for dashboard visualization.

## Phase 4: Integration & Security (Status: [PLANNED])
- [ ] **Swagger UI**: Interactive API documentation.
- [ ] **Rate Limiting**: Protection against abuse.
- [ ] **Structured Logging**: Production-grade logs.
